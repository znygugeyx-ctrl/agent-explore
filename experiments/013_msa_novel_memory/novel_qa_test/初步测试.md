关键发现

1. Easy vs Hard 差异验证了难度分级 — Easy 2.28 vs Hard 1.48
2. 远低于 HotpotQA (4.172) — 符合预期，原因：
  - 小说是中文，MSA 的 prompt 模板和训练都是英文优化的
  - 每节 ~3400 字远大于 HotpotQA 的短文档，chunk 粒度太粗
  - 叙事文本比百科事实更隐晦
3. location 和 item 类型最适合 MSA — 这类信息往往集中在单一段落中
4. reasoning 几乎全部失败 — MSA 的 top-k=16 无法有效覆盖分散在数十节中的因果线索
5. 42% 的题得分为 1 — 说明 MSA 检索到了相关但非正确的文档，模型"看起来在回答"但事实错误
